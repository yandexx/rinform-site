О проекте

Русский Информ (RInform) — это перевод стандартной библиотеки популярной на западе системы Inform 6.

Информ позволяет писать игры с текстовым вводом. Классические примеры таких игр — Adventure и Zork.

Игры

Готовая игра на Информе — это скомпилированный платформонезависимый файл с расширением .z5, .z8 или .zblorb. Чтобы запустить игру, нужно открыть её при помощи одной из программ-интерпретаторов, на выбор:

  • Parchment — самый популярный вариант. Все игры можно запускать онлайн (HTML5).
  • Lectrote — Windows, Linux, OS X (использует Electron).
  • Windows Frotz для Windows.
  • Fizmo — Linux и OS X, поддерживает ncurses и SDL2.
  • Fabularium — Android.

Библиотека

Для разработки игр рекомендуется скачать сборку версии v0.8.3 (обновлена 04 февраля 2018), которая включает в себя компилятор (под Windows), библиотеку, демонстрационные игры и имеющуюся документацию.

Особенности русской версии, которые нужно учесть при написании игр, перечислены в FAQ.

Самая свежая версия библиотеки всегда доступна на репозитарии на BitBucket: https://bitbucket.org/yandexx/rinform. Патчи и баг-репорты сильно приветствуются.

Компиляторы для других операционных систем есть на IF Archive. Компилятор можно собрать самостоятельно из исходников c GitHub (чистый си).
